# 🇳🇵 Nepal E-Governance Portal

A full-stack web application that digitizes government service applications.

## 🔥 Features
- User registration with OTP verification
- Secure login system
- Admin dashboard
- File upload (PDF)
- Application approval/rejection
- Email notifications

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS
- SMTP (Email)
